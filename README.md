ONE-STEP INSTALL
----------------

Use curl (for Mac OS X):

	curl -o - https://raw.github.com/billcava/vimrc/master/auto-install.sh | sh

or wget (for most UNIX platforms):

	wget -O - https://raw.github.com/billcava/vimrc/master/auto-install.sh | sh
